### Setting Up
1. `cd pa-2`
2. `catkin_make`
3. `source devel/setup.bash`
4. `roscore`
5. `chmod +x src/blanchet-pa-2/scripts/pa2.py` (if not using WSL)

### Running
  * `roslaunch blanchet-pa-2 pa2.launch`
